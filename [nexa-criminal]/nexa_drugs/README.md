# nexa_drugs

`nexa_drugs` ist eine abstrakte, serverautoritative Foundation fuer Grow Sites, Batches, Verarbeitung, Qualitaet und Verpackung.

Die Resource enthaelt keine realen Herstellungsanleitungen, keine Rezeptdetails und keine Cliententscheidung ueber Qualitaet oder Output. Verarbeitung verweist kontrolliert auf `nexa_crafting`, Item-Definitionen auf `nexa_items` und Besitz/Output auf `nexa_inventory`.
