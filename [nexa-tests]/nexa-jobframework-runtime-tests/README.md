# nexa-jobframework-runtime-tests

Development-only Runtime-Harness fuer `nexa_jobframework` und `nexa_legaljobs`.

Command:

```text
nexa_test_jobframework_runtime [suite]
```

Suites:

- definitions
- sessions
- phases
- tasks
- progress
- groups
- resource_nodes
- rewards
- mining
- farming
- fishing
- delivery
- trucking
- taxi
- garbage
- mechanic
- security
- restart
- all

Der Command ist fuer Konsole oder ACE `nexa.tests.jobframework_runtime` gedacht. Live-FXServer-Tests muessen offen markiert werden, wenn kein Server laeuft.
