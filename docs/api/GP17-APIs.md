# GP17 APIs

## Phone

Callbacks: `nexa:phone:cb:getSnapshot`, `saveNote`, `sendMessage`, `addContact`, `addGroup`, `logCall`, `savePreferences`, `emergencyCall`.

## Radio

Exports/Callbacks: `RegisterChannel`, `ListChannels`, `JoinChannel`, `LeaveChannel`, `SetPriority`.

## Documents

Exports/Callbacks: `CreateDigitalDocument`, `SignDocument`, `ShareDocument`, plus bestehende Document-Type-, Issue-, Revoke- und Validate-Callbacks.

## Banking UI

NUI callbacks: `bankingRefresh`, `bankingTransfer`, `bankingPayInvoice`.

## MDT UI

NUI callbacks: `mdtRefresh`, `mdtSearchPerson`, `mdtCreateReport`.

## Dispatch UI

NUI callbacks: `dispatchRefresh`, `dispatchAssign`, `dispatchStatus`.
