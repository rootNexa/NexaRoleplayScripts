# Payroll Policies

Policies definieren Organisation, Rank, Betrag pro Periode, Intervall, Mindest-Duty-Zeit, maximale Auszahlung, Prorating, Status, Gueltigkeit und Version.

Status: `draft`, `active`, `suspended`, `expired`, `deleted`.

Betraege sind positive Integer. Genau eine aktive Standardpolicy pro Rank und Zeitraum ist erlaubt.
