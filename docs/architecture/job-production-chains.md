# Job Production Chains

Production Chains verbinden Sammeln, Transport, Verarbeitung, Verpackung, Lieferung und Verkauf.

Verarbeitung nutzt `nexa_crafting`. Direkte Output-Erzeugung ausserhalb von Inventory/Crafting ist verboten. Qualitaet kann spaeter Inputqualitaet, Werkzeuge und Stationen beruecksichtigen.
