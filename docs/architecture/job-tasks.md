# Job Tasks

Tasktypen:

- go_to
- interact
- collect_item
- deliver_item
- process_item
- load_vehicle
- unload_vehicle
- drive_route
- transport_passenger
- repair_vehicle
- inspect_vehicle
- wait
- use_station
- complete_checkpoint
- custom

Tasks besitzen Zieldefinition, Menge, Progress Policy, Validation Policy und optionalen Reward Fragment. Completion ist nie frei vom Client setzbar.
