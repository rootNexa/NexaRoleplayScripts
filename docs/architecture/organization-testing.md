# Organization Testing

Statische Tests pruefen Foundation, Security, Duty, Creator und Runtime-Harness.

Runtime-Suites:

- organizations
- ranks
- memberships
- duty
- economy
- storages
- garages
- modules
- creator
- security
- restart

Live-Tests brauchen FXServer, Datenbank und isolierte Testcharacters. Nicht ausgefuehrte Live-Tests werden als offen dokumentiert.
