# Property Types

Initial registered types:

- `rental_apartment`
- `owned_apartment`
- `house`
- `villa`
- `business_building`
- `garage`

Prepared types:

- `motel`
- `hotel`
- `warehouse`
- `office`
- `shop`
- `land`

Each type defines rentability, purchasability, resident limits, key limits, storage, wardrobe, garage, furniture, alarm and burglary support.
