# Organization Memberships

Memberships sind characterbezogen. Ein Character darf genau eine aktive primaere Organisation besitzen.

## Status

- `invited`
- `active`
- `suspended`
- `left`
- `removed`
- `fired`

Invitations besitzen TTL und koennen nur durch den eingeladenen Character angenommen werden. Entfernen, Befordern, Degradieren und Suspendieren brauchen Hierarchiepruefung und Audit.
