# Organization Ranks

Ranks sind organisationsspezifisch. Position `1` ist der niedrigste Rang, die hoechste Position ist fuehrungsnah. Genau ein Rank ist Owner-Rank.

## Regeln

- Minimum 5 Ranks.
- Maximum 15 Ranks.
- Rank-Key eindeutig innerhalb der Organisation.
- Owner-Rank kann nicht unkontrolliert geloescht werden.
- Niedrigere Ranks duerfen hoehere nicht veraendern.
- Self-Promotion ist verboten.
- Rangpermissions werden serverseitig ausgewertet.

## Permissions

Beispiele: `organization.members.view`, `organization.members.invite`, `organization.members.remove`, `organization.members.promote`, `organization.ranks.manage`, `organization.duty.use`, `organization.account.view`, `organization.storage.modify`, `organization.garage.use`, `organization.dispatch.use`.
