# Organization Types

Starttypen:

- `police`
- `ems`
- `government`
- `gang`
- `business`
- `media`
- `taxi`
- `mechanic`
- `custom`
- `security`
- `fire_department`

Typen definieren, ob Duty erforderlich ist, ob ein Organisationskonto erlaubt ist, ob Storage, Garage, Armory, Dispatch, MDT, Radio, Billing und Payroll vorbereitet sind und ob eine Organisation illegal ist.

Unbekannte Typen werden abgelehnt. Typwechsel muessen kontrolliert und auditierbar sein.
