# Crime Types

Crime-Typen: `store_robbery`, `fuel_robbery`, `atm_breakin`, `bank_robbery`, `jewellery_robbery`, `burglary`, `vehicle_theft`, `drug_activity`, `blackmarket`, `fencing`, `laundering`, `custom`.

Jeder Typ definiert Risiko, minimale Reputation, maximale Heat, Toolbedarf, Responderbedarf, Gruppengroesse, Cooldown und Audit-Level.
