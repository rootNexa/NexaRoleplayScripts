# Vehicles Architecture

Die Fahrzeugfoundation ist serverautoritative Grundlage fuer Fahrzeugdefinitionen, Besitz, Spawn, Zustand, Keys, Garagen und Impound.

Grundregeln: keine Legacy-Framework-Bridges, keine direkte Datenbanktreiber-Nutzung, keine Cliententscheidung ueber VIN, Plate, Owner, Keys oder Persistenz.
