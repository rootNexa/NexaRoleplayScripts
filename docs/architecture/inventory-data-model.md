# Inventory Data Model

Tabellen:

- `nexa_inventories`
- `nexa_inventory_items`
- `nexa_inventory_quickslots`
- `nexa_inventory_audit`

Inventare besitzen Typ, Owner, Slotlimit, Gewichtslimit, Status, Version, Ablaufzeit und Metadaten. Items besitzen Itemname, Slot, Menge, Metadaten, Unit-/Totalgewicht, Haltbarkeit, Ablaufzeit, Instance-ID und Version.
