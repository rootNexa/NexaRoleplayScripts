# Admin Troubleshooting

| Symptom | Check |
| --- | --- |
| Action denied | Permission, duty state, target protection, and reason. |
| Ban does not reject | Confirm `nexa_admin` starts before account resolution or reconnect after start. |
| Teleport has no return | Return state may have expired or player disconnected. |
| Noclip remains visually odd | Use `/noclip` again or restart the resource; cleanup event resets state. |
