# Playerstate Troubleshooting

| Symptom | Check |
| --- | --- |
| Spawn not starting | Confirm account and character are ready. |
| Token invalid | Check source and character binding. |
| Fallback spawn used | Last position may be missing or invalid. |
| Player not active | Confirm spawn confirmation reached server. |
| Position ignored | Snapshot may be rate-limited or implausible. |
