# Jobframework Troubleshooting

Haeufige Fehler:

- `JOB_DEFINITION_NOT_FOUND`: Definition existiert nicht oder wurde geloescht.
- `JOB_DEFINITION_NOT_ACTIVE`: Job ist Draft, suspendiert oder deaktiviert.
- `JOB_SESSION_ALREADY_ACTIVE`: Character hat bereits eine aktive Session.
- `JOB_TASK_VALIDATION_FAILED`: Tasktyp, Target oder Item ist ungueltig.
- `JOB_RESOURCE_NODE_DEPLETED`: Node hat keine verfuegbare Menge.

Bei Migrationfehlern zuerst `nexa-core` und Datenbankstatus pruefen.
