# Job Sessions

Eine Session ist ein temporaerer Auftrag. Sie ist nicht identisch mit Organisation, Duty oder Account.

Status:

- created
- starting
- active
- paused
- completed
- failed
- cancelled
- expired
- manual_review

Start prueft Gameplay-ready, Character, Jobstatus, Zugriff, Organisation, Duty, Cooldown, Gruppengroesse, Tools, Fahrzeuge und Idempotency.
