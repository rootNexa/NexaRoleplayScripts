# Job Resource Nodes

Resource Nodes bilden Mining, Farming und Fishing ab.

Node-Typen:

- node
- field
- area
- water_zone
- timed_source
- organization_source

Menge, Respawn, Toolanforderung, Zugriff, Bucket und Anti-AFK werden serverseitig verwaltet. Harvesting verwendet Reservations, damit keine doppelte Ernte entsteht.
