# Organization Storages

Storages sind technische Registrierungen fuer spaetere Inventory-Zugriffe.

Typen: general, armory, evidence, medical, management, locker.

`nexa_organizations` speichert Storage-Key, Organisation, Inventory-ID, Duty-Regel, Permissions und Metadata. Item-Mutationen laufen spaeter nur ueber `nexa_inventory`.
