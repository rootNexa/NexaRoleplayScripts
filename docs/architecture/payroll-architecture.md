# Payroll Architecture

`nexa_payroll` verwaltet Gehaltsrichtlinien, Perioden, Duty-Zeit-Bewertung, Payroll-Runs, Entries, Auszahlung und Audit.

Standardintervall: zwei Stunden, konfigurierbar.

Finanzierung: Organisationen zahlen aus dem eigenen Economy-Konto. Es gibt keine automatische Geldschoepfung und keine automatische Steuer.
