# MDT Backend

Das MDT ist generisch und zeigt Module nach `mdtType`, Organisation, Permissions und aktivierten Modulen. Kernmodule: Personen, Fahrzeuge, Fahndungen, Berichte, Akten, Strafzettel, Beweise, Kennzeichen und Einsatzverwaltung.

MDT trifft keine Gameplay-Entscheidungen. Es liest kontrollierte Backends: Police, Dispatch, Evidence, Licenses, Vehicles, Characters und Reports.
