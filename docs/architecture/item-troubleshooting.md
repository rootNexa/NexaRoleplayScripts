# Item Troubleshooting

- `ITEM_DEFINITION_NOT_FOUND`: Definition existiert nicht oder ist nicht published.
- `ITEM_TYPE_INVALID`: Typ nicht registriert.
- `ITEM_METADATA_INVALID`: Schema oder Payload pruefen.
- `ITEM_HANDLER_NOT_FOUND`: Handler serverseitig registrieren.
- `ITEM_ASSET_SSRF_REJECTED`: URL-Ziel ist verboten.
