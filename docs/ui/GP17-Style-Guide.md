# GP17 UI Style Guide

GP17 nutzt `nexa_theme` fuer Light/Dark Tokens und `nexa_ui_components` fuer gemeinsame Layout-Contracts.

## Anforderungen

- Responsive Layout fuer 1080p, 1440p und 4K.
- Keine Businesslogik im Browser.
- Tabellen, Filter, Tabs und Dialoge folgen den gemeinsamen Komponenten.
- Fokus und Close-Aktionen werden immer ueber sichere NUI callbacks behandelt.
