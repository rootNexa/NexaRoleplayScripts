# nexa_police

`nexa_police` ist eine generische Police-Foundation fuer LSPD, Sheriff, Government und Federal-Integration.

Sie bildet Festnahmen, Handschellen, Eskorte, Durchsuchung, Beschlagnahme, Waffen-, Fahrzeug- und Personenkontrollen serverseitig ab. Es gibt keine feste LSPD-Hauptarchitektur und keine Cliententscheidung ueber Search- oder Seizure-Ergebnisse.
