# nexa_radio

`nexa_radio` ist die serverseitige GP17-Foundation fuer Funkfrequenzen, Organisationskanaele, Verschluesselungsklassen, Mitgliedschaften und Prioritaeten.

Die Resource implementiert kein eigenes Voice-System. Push-to-talk und Voice-Routing werden spaeter ueber eine dedizierte Voice-Integration angebunden.
