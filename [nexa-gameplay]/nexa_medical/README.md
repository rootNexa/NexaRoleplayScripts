# nexa_medical

`nexa_medical` ist die serverautoritative Foundation fuer Verletzungen, Schmerz, Blutung, Bewusstlosigkeit, Tod, Krankenhaus-Respawn, EMS-Behandlung, Stabilisierung, Medikamente und Medical Reports.

Die Resource entscheidet keine UI und fuehrt keine Client-Heilung ungeprueft aus. Clientdaten sind nur Beobachtungen; State, Reports und Behandlungen sind serverseitig.
